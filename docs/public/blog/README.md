# Blog thumbnails

Drop generated article thumbnails here. They're picked up automatically by the
blog cards and post pages.

**Naming:** `<slug>.png` (or `.jpg` / `.jpeg` / `.webp`).

The **slug** is the article's markdown filename minus the `YYYY-MM-DD-` date prefix.

| Article file | Slug | Thumbnail file |
|---|---|---|
| `2026-07-01-why-i-built-grit.md` | `why-i-built-grit` | `why-i-built-grit.png` |
| `2026-07-02-first-app-in-5-minutes.md` | `first-app-in-5-minutes` | `first-app-in-5-minutes.png` |

**Recommended size:** 1200×630 (1.91:1). If no image exists for a post, the card
falls back to a branded gradient placeholder — so you can add thumbnails as you go.
